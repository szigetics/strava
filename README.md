# Soccer Heatmap Strava


Software requirements
---------------

* Python 3.x <br/>
* gpxpy <br/>
* gmplot  <br/>


Preparing your system
--------------

**Download and install Python Modules**<br/>
`pip3 install gpxpy`<br/>
`pip3 install gmplot`<br/>


**Usage**<br/>
`python3 soccerHeatmap.py filename.gpx`<br/>
